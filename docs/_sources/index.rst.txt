techMiner: A Package for Analyzing Bibliographical Data using Python
============================================================================================

**Date**: September 01, 2019  **Version**: 0.0.0

**Binary Installers:** https://pypi.org/project/techMiner

**Source Repository**: https://github.com/jdvelasq/techMiner

**Documentation**: https://jdvelasq.github.io/techMiner/

**techMiner** is an open source (distributed under the MIT license) and friendly-user
package to do bibliographical analyces. The package was developed and tested in Python
version 3.6. It can be installed from the command line using:

``$ pip3 install techMiner``

**techMiner** can be used interactively at the Python’s command prompt, but a better
experience is achieved when IPython or Jupyter’s notebook are used, allows the
user to fully document the analysis and draw conclusions. Due to the design of
the package, it is easy to use techMiner with the tools available in the ecosystem
of open source tools.

**techMiner** is well suited for bibliographical analysis and it can be used to:

* Co-word analysis.

* Co-ocurrence analysis.
